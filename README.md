# Bot Discord Soul Land News World

Bot Discord sederhana untuk game Soul Land News World, dibuat dengan Python menggunakan discord.py.

## Fitur
- `!hello`: Salam dari bot.
- `!news`: Berita terbaru game (simulasi).
- `!info [nama]`: Info karakter (contoh Tang San atau Xiao Wu).

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Buat bot di [Discord Developer Portal](https://discord.com/developers/applications).
3. Salin token bot dan set sebagai environment variable: `DISCORD_BOT_TOKEN=your_token_here`
4. Jalankan: `python bot.py`

## Aturan Discord
- Bot ini mematuhi TOS Discord: Tidak spam, tidak melanggar privasi, hanya interaksi yang diminta.
- Pastikan bot tidak digunakan untuk aktivitas ilegal.
- Untuk rilis publik, host di server yang stabil (misalnya Heroku, Railway) dan monitor penggunaan.

## Pengembangan
Tambahkan fitur lebih lanjut seperti database untuk data karakter atau integrasi API game jika tersedia.
